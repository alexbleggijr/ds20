# Chain

## Basic usage

Mais métodos de uso em combinação com as animações em [Ionic Animations](https://ionicframework.com/docs/utilities/animations#methods)

```ts
  import { Animations, Easing } 'templarios-2.0';

  Animations.chain(animatedElement)
    .duration(duration)
    .play();

  Animations.chain(animatedElement, stagger)
    .duration(duration)
    .play();

  Animations.chain(animatedElement, stagger, limitStagger, limitCount)
    .duration(duration)
    .play();

  Animations.chain(animatedElement)
    .duration(duration)
    .delay(delay)
    .play();

  Animations.chain(animatedElement, stagger, limitStagger, limitCount)
    .duration(duration)
    .delay(delay)
    .play();

  Animations.chain(animatedElement)
    .duration(duration)
    .easing(Easing.EASE_IN)
    .play();
```
