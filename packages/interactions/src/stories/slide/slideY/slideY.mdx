# SlideY

## Basic usage

Mais métodos de uso em combinação com as animações em [Ionic Animations](https://ionicframework.com/docs/utilities/animations#methods)

```ts
  import { Animations, Easing } 'templarios-2.0';

  Animations.slideY(
    animatedElement,
    start,
    startUnit,
    end,
    endUnit,
  )
    .duration(duration)
    .play();

  Animations.slideY(
    animatedElement,
    start,
    startUnit,
    end,
    endUnit,
  )
    .duration(duration)
    .delay(delay)
    .play();

  Animations.slideY(
    animatedElement,
    start,
    startUnit,
    end,
    endUnit,
  )
    .duration(duration)
    .easing(Easing.EASE_OUT)
    .play();
```
