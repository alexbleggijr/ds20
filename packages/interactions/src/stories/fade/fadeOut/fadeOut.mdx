# FadeOut

## Basic usage

Mais métodos de uso em combinação com as animações em [Ionic Animations](https://ionicframework.com/docs/utilities/animations#methods)

```ts
  import { Animations, Easing } 'templarios-2.0';

  Animations.fadeOut(animatedElement)
    .duration(duration)
    .play();

  Animations.fadeOut(animatedElement)
    .duration(duration)
    .delay(delay)
    .play();

  Animations.fadeOut(animatedElement)
    .duration(duration)
    .easing(Easing.EASE_IN)
    .play();
```
